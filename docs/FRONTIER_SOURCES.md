# Frontier Source Notes

Use this as the manual review list behind the profile README. Keep links that change an implementation choice, not every viral model announcement.

## Reasoning and post-training

- [DeepSeek-R1](https://arxiv.org/abs/2501.12948): reinforcement learning signal for reasoning behavior and long chain-of-thought distillation.
- [s1: Simple test-time scaling](https://arxiv.org/abs/2501.19393): budgeted inference-time reasoning as a practical algorithmic lever.
- [Qwen3](https://qwenlm.github.io/blog/qwen3/): hybrid thinking and non-thinking modes as a product-facing model behavior design.

## Agents and tool use

- [Claude 4](https://www.anthropic.com/news/claude-4): extended thinking, tool use, and long-running task behavior as an agent signal.
- [Model Context Protocol](https://www.anthropic.com/news/model-context-protocol): tool and data-source standardization for agent systems.

## Retrieval and memory

- [GraphRAG](https://www.microsoft.com/en-us/research/project/graphrag/): graph-structured retrieval as a move beyond plain chunk search.
- [Contextual retrieval](https://www.anthropic.com/news/contextual-retrieval): chunk-level context augmentation and retrieval quality.

## Inference systems

- [Speculative decoding](https://arxiv.org/abs/2211.17192): draft/verify decoding for inference acceleration.
- [vLLM paged attention](https://arxiv.org/abs/2309.06180): serving architecture for high-throughput LLM inference.

## Multimodal and architecture

- [Llama 4](https://ai.meta.com/blog/llama-4-multimodal-intelligence/): native multimodality and MoE direction.
- [Gemini 3.1 Pro model card](https://deepmind.google/models/model-cards/gemini-3-1-pro): current model capability and safety framing.
